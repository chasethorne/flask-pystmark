Flask-Pystmark
==============

Flask extension for [Pystmark](https://github.com/xsleonard/pystmark), a Postmark API library.

Flask-Pystmark supports Python 2.6, 2.7, and PyPy.

[Read the complete docs](https://flask-pystmark.readthedocs.org)
